# Laboratório Azure SQL 

Este repositório foi criado como parte de um laboratório prático da DIO, com o objetivo de documentar o processo de criação e configuração de uma instância de Banco de Dados SQL na plataforma Microsoft Azure.

## Objetivo
- Criar um Banco de Dados SQL no Azure
- Configurar regras de firewall e acesso
- Conectar via Azure Portal
- Executar comandos SQL básicos

## Tecnologias utilizadas
- Microsoft Azure
- Azure SQL Database
- Git e GitHub

## Aprendizados
- Configuração de acesso via IP
- Diferença entre autenticação SQL e Microsoft Entra ID
- Execução de scripts SQL no Editor de Consultas

## Anotações sobre Azure SQL

- O Azure SQL Database pode usar autenticação via Microsoft Entra ID
- É necessário liberar o IP no firewall para acesso
- As alterações de firewall podem levar alguns minutos para funcionar
- O Editor de Consultas do Azure usa automaticamente a conta Microsoft

## Observação
Este repositório serve como material de apoio para estudos futuros e consultas rápidas sobre Azure SQL.
